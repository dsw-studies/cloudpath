Курсы специализации (★) простыми словами

что это, зачем, где учить бесплатно

★ Architektura Cloud Computing (Sem 4)

Учит, как вообще устроено облако: типы сервисов (IaaS/PaaS/SaaS), регионы и зоны доступности, как считать стоимость, как проектировать отказоустойчивую систему.

Это фундамент всей специализации — без этого остальные ★-курсы будут непонятны.

AWS Cloud Essentials MS Learn: AZ-900 путь YouTube: AWS CP full course

★ Wirtualizacja i konteneryzacja (Sem 4)

Виртуальные машины vs контейнеры, Docker: образы, контейнеры, Dockerfile, docker-compose.

Docker — обязательный навык на 90% вакансий Cloud/DevOps.

Docker Get Started YouTube: Docker для начинающих Killercoda: практика в браузере

★ Narzędzia developerskie NoCode/LowCode (Sem 5)

Сборка приложений через визуальные платформы (Bubble, Power Apps, n8n) — без ручного написания кода.

Полезно для быстрых прототипов и автоматизации задач без разработчика.

YouTube: NoCode обзор n8n.io

★ Programowanie Front-end (Sem 5)

HTML/CSS/JavaScript + фреймворк (React/Vue) — интерфейсы, которые видит пользователь.

Нужно, чтобы понимать полный цикл разработки приложения, даже если специализируешься на облаке.

freeCodeCamp: Web Design react.dev/learn

★ Platformy i usługi chmur obliczeniowych (Sem 6)

Конкретные сервисы AWS/Azure: compute (EC2/VM), storage (S3/Blob), сети (VPC), базы данных (RDS).

Прямая подготовка к AWS Solutions Architect Associate / Azure AZ-104.

AWS Skill Builder MS Learn: Azure

★ Programowanie Back-end (Sem 6)

Серверная логика, REST API, работа с базой данных со стороны сервера (Node.js/PHP/Python).

Облачный инженер часто разворачивает и настраивает именно back-end приложения.

freeCodeCamp: Back-end & APIs Postman: тестирование API

★ Architektura i komunikacja między systemami (Sem 6)

Как отдельные сервисы общаются: REST, message queues (Kafka/RabbitMQ), микросервисы vs монолит.

Ключевая тема для собеседований на позиции Cloud/Backend Engineer.

microservices.io YouTube: очереди сообщений

★ Systemy i obliczenia rozproszone / Blockchain (Sem 7)

Распределённые вычисления: как множество серверов работают как единая система + основы блокчейна.

Даёт понимание отказоустойчивости и консенсуса — применимо и вне блокчейна (Kubernetes, распределённые БД).

YouTube: Distributed Systems

★ Uczenie maszynowe w chmurach (Sem 7)

Как обучать и разворачивать модели машинного обучения на облачных платформах (AWS SageMaker, Azure ML, GCP Vertex AI).

Соединяет твою специализацию Cloud с ИИ — самый быстрорастущий сегмент рынка труда 2026-2030.

AWS SageMaker docs MS Learn: Azure ML Coursera: ML (Andrew Ng)

★ Bezpieczeństwo i monitoring aplikacji chmurowych (Sem 7)

Защита облачных приложений: IAM (управление доступом), шифрование, логирование, алерты, инструменты типа CloudWatch/Prometheus/Grafana.

Безопасность — обязательный навык, а не опция, для любого Cloud Engineer.

Grafana tutorials AWS Security
