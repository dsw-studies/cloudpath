Синтаксис языков программирования

базовый конспект: переменные, условия, циклы, функции
Это шпаргалка по основам, а не полный курс — держи её открытой рядом, когда пишешь первый код на новом языке. Для практики: пиши тот же самый пример (калькулятор, список задач) на каждом языке по очереди.

🐍 Python

Отступы вместо { } — код внутри блока сдвинут пробелами.

# переменная и типы
name = "Alex"
age = 20
is_student = True

# условие
if age >= 18:
    print("совершеннолетний")
else:
    print("несовершеннолетний")

# цикл
for i in range(5):
    print(i)

# функция
def greet(name):
    return f"Привет, {name}!"

# класс (ООП)
class Student:
    def __init__(self, name):
        self.name = name
    def say_hi(self):
        print(f"Я {self.name}")

☕ Java

Строго типизирован, всё внутри классов, блоки в { }.

// переменная
int age = 20;
String name = "Alex";
boolean isStudent = true;

// условие
if (age >= 18) {
    System.out.println("совершеннолетний");
} else {
    System.out.println("несовершеннолетний");
}

// цикл
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

// класс и метод
class Student {
    String name;
    Student(String name) { this.name = name; }
    void sayHi() { System.out.println("Я " + name); }
}

🗄️ SQL

Язык запросов к базе данных — не программирование в обычном смысле, а декларативные команды.

-- создание таблицы
CREATE TABLE students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
);

-- вставка данных
INSERT INTO students (id, name, age) VALUES (1, 'Alex', 20);

-- выборка
SELECT name, age FROM students WHERE age >= 18 ORDER BY name;

-- изменение
UPDATE students SET age = 21 WHERE id = 1;

-- соединение таблиц
SELECT s.name, c.title FROM students s
JOIN courses c ON s.id = c.student_id;

⚡ JavaScript

Делает страницу интерактивной — работает в браузере и на сервере (Node.js).

// переменные
let age = 20;
const name = "Alex";

// условие
if (age >= 18) {
    console.log("совершеннолетний");
} else {
    console.log("несовершеннолетний");
}

// цикл
for (let i = 0; i < 5; i++) {
    console.log(i);
}

// функция
function greet(name) {
    return `Привет, ${name}!`;
}

// функция-стрелка + массив
const students = ["Alex", "Kate"];
students.forEach(s => console.log(s));

📱 Kotlin (Android)

// переменные: val — неизменяемая, var — изменяемая
val name = "Alex"
var age = 20

// условие
if (age >= 18) println("совершеннолетний") else println("несовершеннолетний")

// цикл
for (i in 0..4) println(i)

// функция
fun greet(name: String): String = "Привет, $name!"

// класс
class Student(val name: String) {
    fun sayHi() = println("Я $name")
}

⚙️ C++

Низкоуровневый контроль над памятью, компилируется напрямую в машинный код.

#include <iostream>
using namespace std;

int main() {
    int age = 20;
    string name = "Alex";

    if (age >= 18) {
        cout << "совершеннолетний" << endl;
    } else {
        cout << "несовершеннолетний" << endl;
    }

    for (int i = 0; i < 5; i++) {
        cout << i << endl;
    }
    return 0;
}

🖥️ PHP

Серверный язык, изначально для веб-страниц — до сих пор держит значительную долю back-end рынка.

<?php
$name = "Alex";
$age = 20;

if ($age >= 18) {
    echo "совершеннолетний";
} else {
    echo "несовершеннолетний";
}

for ($i = 0; $i < 5; $i++) {
    echo $i;
}

function greet($name) {
    return "Привет, $name!";
}
?>

🐚 Bash

Скрипты для Linux-терминала — автоматизация рутинных задач.

#!/bin/bash
name="Alex"
age=20

# условие
if [ $age -ge 18 ]; then
    echo "совершеннолетний"
else
    echo "несовершеннолетний"
fi

# цикл
for i in 1 2 3 4 5; do
    echo $i
done

# функция
greet() {
    echo "Привет, $1!"
}
greet "Alex"
